# 功能
输入一个字，程序会帮你组词。

# 使用方法
在输入框输入你想组的词
![图1](https://cdn.luogu.com.cn/upload/image_hosting/zo05m4qm.png)
点击```组词```或按下```Enter```
![图2](https://cdn.luogu.com.cn/upload/image_hosting/3md5vqua.png)
程序就会帮你组词并显示在输出框
![](https://cdn.luogu.com.cn/upload/image_hosting/v8bqgzt8.png)
